Θ(n²) Tight Bound Visualization

f(n) = 6n² + 100n + 300  ·  c₁ = 5, c₂ = 7, n₀ = 103
f(n) = 6n² + 100n + 300
c₁·n² = 5n²  (lower bound)
c₂·n² = 7n²  (upper bound)
n₀ = 103  (threshold)
Zoom:
✗  Before n₀ = 103

f(n) exceeds the upper bound 7n². The extra terms 100n + 300 are still large enough to push f(n) above c₂·n². The sandwich does not yet hold — hover n < 103 to see this.

✓  After n₀ = 103

f(n) is permanently sandwiched between 5n² and 7n². As n grows, n² dominates and the lower-order terms become negligible — exactly what Θ notation captures.

Formal statement:  ∃ c₁ = 5, c₂ = 7, n₀ = 103 such that
5n²  ≤  6n² + 100n + 300  ≤  7n²   for all n ≥ 103
∴  f(n) = Θ(n²)
